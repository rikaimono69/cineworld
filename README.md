CineWorld Registration Form

1. Обзор Проекта
Проект представляет собой разработку адаптивной формы регистрации для онлайн-платформы CineWorld, выполненную в рамках задания СДМ по направлению Front-end Разработка.  
    Макет Figma: https://www.figma.com/design/eJpXrdN6Q4Air1gfHZfWSq/SDM---Frontend--Copy-?node-id=464-65&t=uU0dGMxZ2sU6lyKK-1   
    Репозиторий: rikaimono69/cineworld
    GitHub Pages: https://rikaimono69.github.io/cineworld

2. Технический Стек
    Языки - HTML5, CSS3, JavaScript
    Библиотеки - air-datepicker

3. Инструкция по Запуску
- Требования
Для запуска проекта необходим установленный Node.js и менеджер пакетов (npm или Yarn).

- Клонирование и Установка
    1.  Клонируйте репозиторий:
    git clone https://github.com/rikaimono69/cineworld.git
    cd cineworld

    2. Установите зависимости проекта (в частности, air-datepicker):
    npm install
    Если используете Yarn
    yarn install
    3. Запуск
    Так как проект не использует локальный сервер для сборки (чистый HTML/CSS/JS), для запуска достаточно:
        1. Открыть файл index.html в вашем браузере.
        2. Если вы используете VS Code, воспользуйтесь расширением Live Server.
    Проект станет доступен на локальном хосте.

4. Ключевые Требования Задания
- Код и Стили: Вёрстка адаптивна, соответствует семантике, выполнена по методологии БЭМ.
- Именование JS: Переменные/функции названы согласно lowerCamelCase.
- Календарь: Реализован с использованием air-datepicker с оверлеем и блокировкой скролла.
- Валидация: Реализована индивидуальная и общая валидация, кнопка REGISTER блокируется при ошибках.
- Тёмная Тема: Реализована (в theme.js) с сохранением выбора в localStorage.
